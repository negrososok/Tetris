# Курсова робота: Гра "Тетріс" на C++ з використанням SFML

Цей проєкт є практичною реалізацією гри "Тетріс" мовою C++ з використанням мультимедійної бібліотеки SFML (Simple and Fast Multimedia Library). 

Програма має спрощену структуру коду без використання складних об'єктно-орієнтованих шаблонів проектування, що робить її легкою для розуміння та аналізу. Усі ігрові змінні та стан гри винесені в глобальну область видимості, а логіка розділена на прості функції.

## Структура проєкту

Проєкт містить такі файли та папки:
* main.cpp — вихідний код програми (ігрова логіка, рендеринг графіки, обробка подій, відтворення звуків).
* CMakeLists.txt — конфігураційний файл для збирання проєкту через CMake.
* Tetris.exe — готовий скомпільований виконуваний файл гри для ОС Windows.
* sfml/ — заголовні файли та бібліотеки SFML.
* sounds/ — папка з аудіоресурсами (автентична ретро-музика та звукові ефекти у форматі MP3).
* *.dll — динамічні бібліотеки SFML, необхідні для запуску гри.

## Опис логіки програми

### 1. Збереження стану гри
Стан гри контролюється трьома логічними змінними типу bool:
* showMenu — відображення головного меню гри.
* isPaused — режим павзи (зупиняє ігровий таймер та фонову музику).
* isGameOver — стан закінчення гри.

### 2. Ігрова сітка та фігури
* Ігрове поле має розмір 10 стовпців на 20 рядків (задано константами COLS та ROWS).
* Координати блоків поточної та наступної фігур зберігаються у масивах структур Point з полями x та y.
* Обертання фігур виконується за стандартною формулою повороту точки на 90 градусів за годинниковою стрілкою навколо центру фігури. Квадратна фігура (O-тип) не обертається.
* Для запобігання виходу фігури за межі поля при обертанні біля стін реалізована спрощена система Wall Kick (зсув на 1 блок ліворуч або праворуч).

### 3. Рівні та швидкість гри
* Новий рівень нараховується за кожні 5 очищених ліній (формула: level = lines / 5).
* З підвищенням рівня зменшується затримка часу між падіннями фігур (гра стає швидшою). На початкових рівнях затримка розраховується як 1.0 - level * 0.09 секунд.

### 4. Нарахування очок
За очищення ліній нараховуються очки за класичною системою, помноженою на коефіцієнт поточного рівня (level + 1):
* 1 лінія — 40 очок.
* 2 лінії — 100 очок.
* 3 лінії — 300 очок.
* 4 лінії (Тетріс) — 1200 очок.

### 5. Аудіосистема
Гра використовує двоканальний вивід звуку:
* Окремий потік для фонової музики (sf::Music). У меню грає зациклена тема, а під час гри автоматично працює плейлист із 3 оригінальних ретро-треків, які перемикаються по колу після завершення кожного треку.
* Окремий канал для коротких звукових ефектів (sf::Sound). Під час програвання звуку підвищення рівня (levelup) інші звукові ефекти тимчасово приглушуються, щоб запобігти їхньому накладанню.

## Керування у грі

### У головному меню:
* W / S або Стрілки вгору / вниз — вибір пункту меню.
* Enter — підтвердження вибору (запуск гри або вихід).
* Escape — вихід з програми.

### Під час гри:
* A / D або Стрілка вліво / вправо — рух фігури вбік.
* W або Стрілка вгору — обертання фігури.
* S або Стрілка вниз — прискорене падіння фігури (Soft Drop).
* Space (Пробіл) — миттєве падіння та фіксація фігури (Hard Drop).
* P або Escape — пауза.
* R — почати раунд спочатку (Restart).
* M — повернутися в головне меню (працює під час паузи або на екрані закінчення гри).

## Як скомпілювати проєкт

Для компіляції проєкту на Windows за допомогою компілятора GCC (MinGW) та встановленою бібліотекою SFML використовуйте таку команду:

g++ -O3 main.cpp -o Tetris.exe -I sfml/include -L sfml/lib -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio
