# Курсова робота: Гра "Тетріс" на C++ з використанням SFML

Цей проєкт є практичною реалізацією класичної гри "Тетріс" мовою C++ з використанням мультимедійної бібліотеки SFML (Simple and Fast Multimedia Library). 

Програма написана у простому студентському стилі без використання складних об'єктно-орієнтованих шаблонів проектування, що робить її легкою для розуміння та аналізу. Усі ключові змінні та стан гри винесені в глобальну область видимості, логіка розділена на зрозумілі функції, а коментарі в коді лаконічні (по 1-2 слова українською мовою).

## Структура проєкту

Проєкт містить такі файли та папки:
* main.cpp — вихідний код програми (ігрова логіка, рендеринг графіки, обробка подій, відтворення звуків та консольне логування).
* CMakeLists.txt — конфігураційний файл для збирання проєкту через CMake.
* Tetris.exe — готовий скомпільований виконуваний файл гри для ОС Windows.
* sounds/ — папка з аудіоресурсами (фонова музика та звукові ефекти у форматі MP3).

## Опис логіки програми

### 1. Збереження стану гри
Стан гри контролюється логічними змінними типу bool:
* `pokazatyMenu` — відображення головного меню гри.
* `naPauzi` — режим паузи (зупиняє ігровий таймер та фонову музику).
* `kinetsHry` — стан закінчення гри (Game Over).

### 2. Ігрова сітка та фігури
* Ігрове поле має розмір 10 стовпців на 20 рядків (задано константами `KOLONKY` та `RYADKY`).
* Координати блоків поточної та наступної фігур зберігаються у масивах структур `Tochka` з полями `x` та `y` (`potochna[4]` та `nastupna[4]`).
* Обертання фігур виконується за стандартною математичною формулою повороту точки на 90 градусів навколо центру фігури (`centr = temp[1]`). Квадратна фігура не обертається.
* Для запобігання виходу фігури за межі поля при обертанні біля стін реалізована система зсувів (Wall Kick).

### 3. Рівні та швидкість гри
* Новий рівень нараховується за кожну 5-ту очищену лінію (`riven = linii / 5`).
* З підвищенням рівня зменшується затримка часу між падіннями фігур (гра стає швидшою).

### 4. Нарахування очок
За очищення ліній нараховуються очки за класичною схемою, помноженою на коефіцієнт рівня (`riven + 1`):
* 1 лінія — 40 очок.
* 2 лінії — 100 очок.
* 3 лінії — 300 очок.
* 4 лінії (Тетріс) — 1200 очок.

### 5. Аудіосистема
Гра використовує двоканальний вивід звуку:
* Окремий потік для фонової музики (`sf::Music`). У меню грає зациклена тема, а під час гри автоматично працює плейлист із 3 оригінальних ретро-треків, які перемикаються по колу після завершення кожного треку.
* Окремий канал для коротких звукових ефектів (`sf::Sound`).

### 6. Консольне логування (Console Logging)
У програму додано систему консольного логування (`zapysaty_loh`). Оскільки гра скомпільована як консольний застосунок, при її запуску автоматично відкривається вікно консолі, куди в реальному часі виводяться логи:
* Старт ігрової сесії.
* Кількість очищених ліній та поточний рахунок.
* Підвищення рівня складності.
* Фінальні результати на екрані Game Over.

Це дозволяє здійснювати діагностичне тестування та відстежувати правильність роботи ігрової логіки прямо під час гри.

## Керування у грі

### У головному меню:
* W / S або Стрілки вгору / вниз — вибір пункту меню.
* Enter — підтвердження вибору (запуск гри або вихід).
* Escape — вихід з програми.

### Під час гри:
* A / D або Стрілка вліво / вправо — рух фігури вбік.
* W або Стрілка вгору — обертання фігури.
* S або Стрілка вниз — прискорене падіння фігури (Soft Drop).
* Space (Пробіл) — миттєве падіння та фіксація фігури (Hard Drop).
* P або Escape — пауза.
* R — почати раунд спочатку (Restart).
* M — повернутися в головне меню (працює під час паузи або на екрані закінчення гри).

## Як скомпілювати проєкт

Для компіляції проєкту на Windows використовуйте один із наведених нижче способів у терміналі (VS Code, PowerShell або звичайний CMD).

### Варіант 1: Універсальна однорядкова команда (підходить для CMD та PowerShell)
Замініть `C:/шлях_до_SFML` на ваш реальний шлях до папки з бібліотекою:
```bash
g++ -std=c++17 -O2 main.cpp -I"C:/шлях_до_SFML/include" -L"C:/шлях_до_SFML/lib" -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio -o Tetris.exe
```

### Варіант 2: Через змінну (тільки для PowerShell)
```powershell
# Вкажіть ваш власний шлях до папки з бібліотекою SFML
$sfml = "C:/шлях_до_SFML"

# Компіляція програми
g++ -std=c++17 -O2 main.cpp -I"$sfml/include" -L"$sfml/lib" -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio -o Tetris.exe
```
