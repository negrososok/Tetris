# Курсова робота: Розробка гри "Тетріс" мовою C++ з бібліотекою SFML

Цей проєкт є готовою практичною частиною курсової роботи з теми **"Розробка гри Тетріс"**. Програма написана на мові C++ з використанням кросплатформної мультимедійної бібліотеки **SFML** (Simple and Fast Multimedia Library).

Код структурований максимально просто, зрозуміло та лаконічно, без зайвих складних архітектурних рішень (класів-одинаків, фабрик, складних менеджерів пам'яті тощо), що дозволить вам з легкістю захистити роботу та відповісти на будь-які запитання викладача. Коментарі у коді є гранично короткими (1–2 слова) та вказують лише на логічні блоки програми.

---

## 🏗️ Структура проєкту

Проєкт складається з таких основних файлів:
1.  `main.cpp` — увесь спрощений вихідний код гри, включаючи ігрову логіку, керування клавіатурою, систему нарахування очок, рендеринг графіки та аудіоплеєр.
2.  `CMakeLists.txt` — файл конфігурації для системи автоматичного збирання CMake.
3.  `README.md` — цей документ (опис архітектури, інструкція та промпти для ElevenLabs).
4.  `Tetris.exe` — готовий скомпільований виконуваний файл гри.
5.  `sfml/` — папка з бібліотекою SFML (заголовки та бібліотеки).
6.  `*.dll` файли — динамічні бібліотеки, потрібні для запуска гри на Windows (включаючи `sfml-audio-2.dll` та `openal32.dll`).
7.  `generate_sounds.py` — Python-скрипт для автоматичного завантаження музики та генерації звуків через ElevenLabs.

## 📖 Як працює код: детальний опис для захисту

Нижче наведено простий розбір логіки, яка закладена у файлі `main.cpp`:

### 1. Державна логіка гри (State Logic)
Гра використовує три прості логічні змінні (`bool`), зрозумілі кожному студенту та викладачу:
*   `showMenu` — якщо `true`, відображається Головне меню.
*   `isPaused` — якщо `true`, гра на павзі (зупиняє ігровий таймер та фонову музику).
*   `isGameOver` — якщо `true`, гра завершена (активує червоний екран програшу).

### 2. Людяний дизайн головного меню
Дизайн меню повністю стилізований під оригінальний Тетріс:
*   Назва **T E T R I S** виводиться великими літерами, кожна з яких пофарбована в окремий яскравий колір із палітри кольорів ігрових фігур.
*   На тлі меню малюються красиві статичні декоративні фігурки Тетрісу (T, L, I) в кутках екрана.
*   Замість простого текстового покажчика навколо активної кнопки меню малюється красива підсвічена бірюзова рамка-плашка. Колір тексту активної кнопки стає білим, а неактивної — тьмяно-сірим.

### 3. Двоканальна аудіосистема та Playlist
*   **Музика головного меню (`menuMusic`)**: Окремий звуковий потік, який автоматично запускається та грає із зацикленням, поки `showMenu == true`. Як тільки гравець запускає гру, музика меню зупиняється.
*   **Автоматичний ігровий плейлист (`music`)**: Містить масив із 3 різних ретро-композицій (`theme1.mp3`, `theme2.mp3`, `theme3.mp3`).
    *   При старті гри випадковим чином обирається один із 3 треків для різноманітності.
    *   У циклі програми постійно перевіряється стан плеєра. Як тільки поточна композиція добігає кінця (`music.getStatus() == sf::SoundSource::Stopped`), плеєр автоматично завантажує та вмикає наступну пісню за колом.
*   **Повернення в меню**: На екрані павзи або програшу додано можливість натиснути клавішу `M`, яка повертає гравця в головне меню та знову вмикає спокійну музику меню.

### 4. Логіка зіткнень (`check_collision`)
Функція `check_collision` приймає поточні координати фігури на полі та масив сітки. Вона перевіряє кожну з 4 точок фігури:
1.  Чи не виходить точка за ліву/праву межу ігрового поля (`x < 0` або `x >= 10`).
2.  Чи не впала фігура нижче дна поля (`y >= 20`).
3.  Чи не перетинається вона з уже зафіксованими блоками у масиві `grid` (`grid[y][x] != 0`).
Якщо хоча б одна умова виконується, функція повертає `true`.

### 5. Обертання фігур та система Wall Kick
Для обертання використовується формула обертання на 90 градусів за годинниковою стрілкою навколо опорної точки фігури:
*   `new_x = center.x - (old_y - center.y)`
*   `new_y = center.y + (old_x - center.x)`
*   **Виняток**: Квадратна фігура (O-тип) не обертається.
*   **Система Wall Kick (відштовхування від стін)**: Якщо після повороту фігура виявляється за межами поля, гра намагається змістити її на 1 блок ліворуч або праворуч.

## 🎮 Керування у грі

### У Головному Меню (`showMenu == true`)
*   **W** / **S** або **Стрілки вгору/вниз** — переміщення вибору між "START GAME" та "EXIT".
*   **Enter** — підтвердження вибору.
*   **Escape** — вихід із гри.

### Під час гри
*   **A** / **← (Стрілка вліво)** — Рух фігури ліворуч.
*   **D** / **→ (Стрілка вправо)** — Рух фігури праворуч.
*   **W** / **↑ (Стрілка вгору)** — Обертання фігури за годинниковою стрілкою.
*   **S** / **↓ (Стрілка вниз)** — М'яке скидання (Soft Drop) — прискорене падіння.
*   **Space (Пробіл)** — Жорстке скидання (Hard Drop) — миттєве падіння та фіксація.
*   **P** / **Escape** — Павза (Pause).
*   **M** — Повернутися в Головне меню (можна натиснути під час павзи).
*   **R** — Перезапуск гри (Restart).

### На екрані програшу (`isGameOver == true`)
*   **R** — Перезапустити гру.
*   **M** — Повернутися в Головне меню.

## 🎙️ Автоматична генерація звуків та завантаження музики

Скрипт `generate_sounds.py` повністю автоматизує процес підготовки звукових ресурсів.

### Що робить скрипт:
1.  **Автоматично завантажує 4 безкоштовні ретро-мелодії (OGG)** з відкритих джерел на GitHub та зберігає їх у папку `sounds/` під назвами:
    *   `menu.ogg` (тема для меню)
    *   `theme1.ogg` (тема 1 для гри)
    *   `theme2.ogg` (тема 2 для гри)
    *   `theme3.ogg` (тема 3 для гри)
2.  **Генерує звукові ефекти через ElevenLabs Pro** за оптимізованими текстовими запитами та зберігає їх у папку `sounds/` під назвами:
    *   `move.mp3` (рух фігури та приземлення)
    *   `rotate.mp3` (м'який та приємний ретро-клік обертання)
    *   `clear.mp3` (зникнення лінії)
    *   `levelup.mp3` (перехід на новий рівень)
    *   `gameover.mp3` (мелодія програшу)
3.  **Перевіряє наявність файлів**: Якщо файли вже існують на диску, скрипт пропускає їх завантаження, що дозволяє довантажувати лише відсутні звуки без зайвих витрат лімітів ElevenLabs Pro.

## 🛠️ Як запустити гру

1. Запустіть скрипт для завантаження музики та ефектів:
   ```bash

   ```
2. Введіть ваш API-ключ ElevenLabs Pro для генерації ефектів.
3. Запустіть файл **`Tetris.exe`** подвійним кліком!
